# ShortVideoCrawl

Short video crawler based on [scrapy](https://github.com/scrapy/scrapy), currently supports:

- kuaishou

## Usage

requirements:

- python 3.10
- poetry

### prepare

```bash
git clone https://github.com/dxsooo/ShortVideoCrawl
poetry install
```

### run

For example:

```bash
cd shortvideocrawl
scrapy crawl kuaishou
```
